这是一个基于 [ACL4SSR](https://github.com/ACL4SSR/ACL4SSR) 的自定义配置文件（`luyuyun.ini`），专为 Clash 订阅转换设计。

核心特点是**Emby 流量隔离**：配置中特别设置了 Emby 专属策略组，并确保其他自动选择或地区策略组（如香港、日本、美国等节点）排除 Emby 节点，防止观看 Emby 时误连到不合适的节点或产生冲突。

## ✨ 主要功能

*   **Emby 专属深度适配**：
    *   **开箱即用**：针对鹿语云机场的专属 Emby 服务做了全方位适配，用户**无需额外配置**即可直接使用。
    *   **自动匹配低倍率**：规则自动识别并分流至 **0.1 倍率的 Emby 专属节点**，在享受高清流媒体的同时最大化节省流量。
    *   **流量隔离**：独立的 `🎬 Emby资源` 策略组优先匹配 `tv.luyuyun.com` 及关键字 `emby`。同时，其他策略组（自动选择、地区节点等）通过正则自动排除 Emby 节点，防止观看 Emby 时误连到不合适的节点。
*   **TikTok 稳定运营**：
    *   **绑定家宽节点**：`📹 TikTok` 策略组通过正则 `(?=.*家宽)` 专门筛选并绑定了家宽节点，有效降低风控风险。
    *   **手动选择模式**：采用 `select` 手动选择策略，避免自动切换节点导致的 IP 频繁变动，为 TikTok 运营提供更稳定的网络环境。
*   **精细化策略分组**：
    *   包含 `🚀 节点选择`、`♻️ 自动选择`、`🚀 手动切换` 等基础分组。
    *   针对特定服务的分组：`📲 电报消息`、`💬 Ai平台` (OpenAI/ChatGPT)、`📹 TikTok`、`📹 油管视频`、`🎥 奈飞视频`、`🎮 游戏平台` 等。
    *   **地区自动分组**：自动提取 `🇭🇰 香港`、`🇯🇵 日本`、`🇺🇲 美国`、`🇨🇳 台湾`、`🇸🇬 狮城`、`🇰🇷 韩国` 等地区的节点。
*   **全面的规则集**：
    *   集成 ACL4SSR 维护的去广告 (`🛑 广告拦截`)、应用净化 (`🍃 应用净化`) 规则。
    *   微软、苹果、谷歌、Steam 等服务的直连或代理规则。
    *   国内外媒体智能分流。

## 🛠 使用方法

此配置文件主要配合支持 ACL4SSR 规则的**订阅转换**服务使用（如各类在线订阅转换网站或本地搭建的转换后端）。

1.  **获取配置文件地址**：
    将本项目中的 `luyuyun.ini` 文件内容托管到网络上（如 GitHub Raw 链接），或者直接使用本文件的 Raw 地址。
    
2.  **进行订阅转换**：
    *   打开订阅转换工具（如 Subconverter）。
    *   在“远程配置”或“自定义配置”栏中，填入 `luyuyun.ini` 的在线地址。
    *   输入你的机场订阅链接。
    *   生成新的 Clash 订阅链接。

3.  **导入 Clash**：
    将生成的订阅链接导入 Clash 客户端（Clash for Windows, Clash Verge, ClashX 等）即可使用。

## 📂 文件说明

*   `luyuyun.ini`: 核心配置文件，包含规则集（Rulesets）定义和代理组（Proxy Groups）策略。

## 📝 注意事项

*   配置文件中使用了正则表达式来过滤和匹配节点名称，请确保你的机场节点命名规范，以便规则能正确识别（例如包含“香港”、“美”、“Emby”等关键字）。
*   Emby 策略组默认使用 `url-test` 模式进行测速选择，URL 为 `http://www.gstatic.com/generate_204`。
